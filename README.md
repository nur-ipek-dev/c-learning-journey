# c-learning-journey
My journey learning C programming from scratch

## Day 01 – İlk C Programım

Bu derste C dilinde ilk programımı yazdım.

Öğrendiklerim:
- `#include <stdio.h>` → giriş/çıkış fonksiyonları
- `int main(void)` → programın başlangıç noktası
- Değişken türleri:
  - `int`
  - `float`
  - `double`
  - `char`
- `printf()` ile ekrana yazdırma
- Program sonunda `return 0;`

İlk programım ekrana şu bilgileri yazdırıyor:
- Bir tam sayı
- Bir ondalıklı sayı
- Bir karakter


## Day 02 – Temel Operatörler ve Girdi Alma

Bugün C dilinde temel operatörleri ve kullanıcıdan veri almayı öğrendim.

Öğrendiklerim:
- scanf() ile kullanıcıdan veri alma
- sizeof ile veri tiplerinin bellekte kapladığı alan
- Aritmetik operatörler (+ - * / %)
- Atama operatörleri (= += -= *= /=)
- Karşılaştırma operatörleri (== != > < >= <=)
- Mantıksal operatörler (&& || !)

Not:
- Karşılaştırma ve mantıksal operatörler 1 (true) veya 0 (false) döndürür.
